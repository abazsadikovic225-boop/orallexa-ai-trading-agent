## 🖥️ What this app does

orallexa-ai-trading-agent is a Windows app that helps you follow trading signals and review market ideas in one place. It uses several AI agents to compare signals, discuss the case, and then pass the result to a portfolio manager step.

It brings together:

- prediction market data from Polymarket and Kalshi
- signal fusion from many sources
- ML model output from multiple trained models
- a debate flow with Bull, Bear, and Judge roles
- a final portfolio gate before action

The app is built for end users who want a clear view of market signals without handling the technical side.

## 🧠 How the agent flow works

The app uses a multi-agent chain so each step checks the next step.

### Signal fusion
The system gathers signals from many sources and blends them into one view.

### Bull agent
The Bull agent makes the case for a trade. It looks for reasons to enter.

### Bear agent
The Bear agent argues against the trade. It looks for weak points, gaps, and risks.

### Judge agent
The Judge reads both sides and decides which case is stronger.

### Portfolio Manager
The Portfolio Manager checks the final result against your limits before the app moves ahead.

This setup helps the app avoid one-sided decisions.

## 🛠️ Troubleshooting

### The app does not open
- Check that the file finished downloading
- Right-click the file and choose Run as administrator
- If the app came in a zip file, extract it first
- Try downloading the newest release again

### The app opens and closes right away
- Restart Windows
- Make sure your antivirus did not block the file
- Reinstall from the latest release page

### No signals appear
- Check your internet connection
- Confirm that your data source keys are correct
- Refresh the feed inside the app
- Wait a few minutes and try again

### The app feels slow
- Close other large apps
- Use fewer live sources at once
- Run in paper mode first
- Give the system a few minutes after launch

### A source shows an error
- Open settings
- Turn the source off and on again
- Re-enter the key if needed
- Check that the provider is online
